# jul-sh Claude Code Plugins

A Claude Code plugin marketplace by Juliette Pluto.

## Plugins

- **[captain-codex](https://github.com/jul-sh/captain-codex)** — Codex plans, Claude implements, Codex reviews.
- **[claude-caffeinate](https://github.com/jul-sh/claude-caffeinate)** — Prevents your Mac from sleeping while Claude Code is actively working.

## Installation

```
/plugin marketplace add jul-sh/claude-plugins
```
